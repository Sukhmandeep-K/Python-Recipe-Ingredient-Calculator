# Python Recipe Ingredient Calculator

## Description
A Python program that calculates the total ounces needed for each ingredient based on
user-entered ingredient amounts and the number of servings.

## Features
- Allows users to enter ingredient names and ounce amounts
- Accepts a user-defined number of servings
- Calculates total ingredient quantities using arithmetic
- Displays clear and readable output

## Skills Practiced
- Python
- User input handling
- Arithmetic calculations
- Output formatting

## How to Run
1. Open the file in a Python environment such as CodeHS.
2. Run the program.
3. Enter ingredient names and ounce amounts when prompted.
4. Enter the number of servings to calculate total ounces.

## Example Output
Enter ingredient 1: Flour
Ounces of Flour: 2
Enter ingredient 2: Sugar
Ounces of Sugar: 1
Enter ingredient 3: Butter
Ounces of Butter: 0.5
Numbers of servings: 4

Total ounces of Flour: 8.0
Total ounces of Sugar: 4.0
Total ounces of Butter: 2.0
